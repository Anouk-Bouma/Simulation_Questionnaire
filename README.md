# Simulation_Questionnaire

This repository contains all coding files used for the analysis in project: https://osf.io/6r2eu/overview

Due to anonymity concerns, data is not shared for this project.

## Files in repository
*data_cleaning.qmd* contains all cleaning steps that were taken to clean and anonymize the data.

*data_analysis.qmd* contains all data analysis steps.

*analysis_journal_metadata.qmd* contains the analysis used to produce Table 1 in our paper.
